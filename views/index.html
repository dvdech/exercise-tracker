Exercise tracker

you can use the UI below for two the endpoints.

the remaining endpoints can be referenced below.

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

Endpoints

  • GET /api/users/:_id/logs
    • used to get a full exercise log of a user using their 'id' pass in the id to the URI where it specifies ':_id' you can add the following query parameters to the end of the endpoint: from, to = dates (yyyy-mm-dd); limit = number
  • GET /api/users/:_id/exercises
    • used to get all exercises associated with a user using their 'id' pass in the id to the URI where it specifies ':_id'
  • GET /api/users
    • used to get an array of all users created in the database
  • POST /api/users
    • used to create a user pass data using raw body payload | content-type application/json
  • POST /api/users/:_id/exercises
    • used to add an exercise to the specified user using their 'id' pass data using raw body payload | content-type application/json
